Desafio Pratico Rocketseat
Aplicando conhecimentos basicos em Pyhton apos as aulas assistidas na plataforma
Desafio: fazer um programa de lista de tarefas, com as opções: 
    1. Adicionar Tarefa
    2. Ver Tarefas
    3. Atualizar Tarefa
    4. Completar Tarefa
    5. Deletar Tarefas Completadas
    6. Sair
